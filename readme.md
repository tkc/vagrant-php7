# Laravel vagrant-php7

## download package.box
https://drive.google.com/file/d/0Bwb7S_-TeW_eT0NkVnZ6UU9xWUk/view?usp=sharing

## Box add
```
$vagrant box add  laravel/php7 package-laravel-php7-postgresql.box
```

## Start Vagrant

```
$ vagrant up
$ vagrant ssh
```

## Installed list
```
php7
Nginx
MySQL
PostgreSQl
```
