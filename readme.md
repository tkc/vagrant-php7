# Laravel vagrant-php7

## download package.box
https://drive.google.com/file/d/0Bwb7S_-TeW_eT0NkVnZ6UU9xWUk/view?usp=sharing

## Box add
```
$ vagrant box add $vagrant box add  package-laravel-php7-postgresql.box laravel/php7
```

## Start vagrant

```
$ vagrant up
$ vagrant ssh
```

## Installed list
```
php7
Nginx
MySQL
PostgreSQl
```
